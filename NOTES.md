# Peter Mac Informatics Consulting Training

Landing page for PMacDaSci training materials.

## Edit checklist

1. Replace every `TODO_CONNECT_PAGE_URL` in `index.html` with the Peter Mac Connect page URL for scheduled workshops.
2. Review the workshop list and add/remove cards as needed.
3. Commit changes to the `main` branch of `PMacDaSci.github.io`.
4. Confirm GitHub Pages is enabled for the repository.
